# Roadmap Archive

This directory stores concise summaries of completed major milestones when historical context would otherwise be difficult to reconstruct from issues, pull requests, and release notes.

## Archive entry contents

A milestone summary should include:

- Milestone name and completion date
- Merged pull requests and final SHAs
- What shipped
- Validation evidence
- What was explicitly deferred
- Follow-up issue links
- Any migration or downstream-upgrade notes

## What does not belong here

- Full changelog duplication
- Temporary implementation notes
- Uncommitted ideas
- Active workstream plans
- Raw CI logs

Use `CHANGELOG.md` for release-facing change history, GitHub issues for active work, and `docs/roadmap/decisions/` for architectural decisions.
